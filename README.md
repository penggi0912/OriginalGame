InGameフォルダ ⇒ Maps ⇒ Game_Start_Menuからゲームを開始できます。<br/>
ゲームに使用中のファイルはすべてInGameフォルダ内で管理しています。<br/>
ゲームパッド推奨。<br/>
<br/>
<br/>
【基本操作（ゲームパッド）】<br/>
Xボタン(□ボタン)：攻撃<br/>
Bボタン(○ボタン)：アクション(武器召喚、インタラクト)<br/>
Aボタン(xボタン)：ジャンプ<br/>
左トリガー1(L1)+XYBAボタンのいずれか：スキル攻撃<br/>
右トリガー1(R1)：ドッジロール　※現在改修中<br/>
右スティック押下(R3)：ロックオン/オフ<br/>
スタートボタン：メニュー表示<br/>
